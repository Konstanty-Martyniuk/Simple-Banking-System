Get the game mode

Imagine your program is a game and it can accept some pairs: parameter and value, like game mode, game level, etc. It also has some parameter values by default. If it has no parameters, the default ones will be used.

Write a program, that accepts String parameter named "mode" (no quotes), and displays its value. Input is a sequence of pairs: parameter name and following value. Note, that the "mode" word can occur just once as a parameter name and multiple times as a parameter value.

If no parameter is provided, it uses the default value: "default" (no quotes).


Sample Input:
level beginner mode battle

Sample Output:
battle


Sample Input:
parameter1 mode parameter2 value2

Sample Output:
default



Memory limit: 256 MB
Time limit: 8 seconds
